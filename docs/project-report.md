# Project Report

## Objective
Analyze terminal efficiency and cargo movement performance.

## Data Model
Star schema:
- fact_cargo_movements
- dim_terminal
- dim_vessel
- dim_time

## Key KPIs
- Total Movements
- On Target %
- At Risk %
- Bottleneck %
- Avg Duration

## Insights
- Bottlenecks are concentrated in specific terminals
- Performance varies by vessel category
- Seasonal patterns impact operations
